# webpack4.x_Demo
基于webpack4的多页应用配置，包括常见的插件使用与配置。
要点都有注释！
如有帮助，请赐我一颗小星星。
功能有代码压缩，scss，less转css，es6转es5，打包出来的js和css都会添加hash值。

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9005
npm run start

# build for production with minification
npm run build

```
